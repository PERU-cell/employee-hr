# Employee HR Management System

A MySQL project to manage employee records, salaries, attendance and leaves.

## Tables
- departments
- employees
- jobs
- salaries
- attendance
- leave_requests

## How to Run
1. Open MySQL Workbench
2. Run schema.sql first
3. Run data.sql to insert data
4. Run queries from queries.sql

## Tools Used
- MySQL 8.0
- MySQL Workbench
